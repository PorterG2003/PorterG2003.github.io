Pokemon

Simulate Pokémon Battles with Ease

This Python-based application lets users simulate battles between Pokémon, manage teams, and view stats. It's a great tool for fans, showcasing skills in database management, and Python programming.

Features

  • Simulate battles between two Pokémon
  • View average stats by Pokémon type
  • Add or remove Pokémon from your team
  • Display top Pokémon by stat
  • Find the most specialized Pokémon for each stat

Installation

Follow the steps in the README.md in the repository to get the project up and running. The application is built with Python, Tkinter, Pillow, and MySQL. It requires certain dependencies to be installed, which can be easily done using the provided requirements.txt file.
